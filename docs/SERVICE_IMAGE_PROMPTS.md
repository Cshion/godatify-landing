# Service Hero Image Generation Prompts for Gemini
**Date:** May 6, 2026 | **Target Dimensions:** 1200×630px (OG image landscape) | **Brand Theme:** Premium Enterprise Consultancy

---

## 🎨 Brand Foundation

### Brand Colors
- **Primary Teal:** #135c51 (deep, grounded, trustworthy)
- **Accent Neutrals:** Off-white, light gray, charcoal
- **Forbidden:** Bright gradients, playful illustrations, startup colorfulness

### Design Language
- Premium consultancy aesthetic (like [DataArt](https://www.dataart.com/es))
- Professional photography + minimal graphic elements
- No stock photos of handshakes or generic corporate imagery
- Clean hierarchy, generous whitespace, deliberate composition
- Confidence without arrogance; authority with approachability

### Emotional Target
**Confianza y Seguridad** — Enterprise decision-makers in LATAM see these visuals and think: "This is serious, expert, I can trust them with enterprise complexity."

## 1️⃣ **Big Data Management** → `bg-bigdata.jpg`

### Service Context
- **Tagline:** "Escalabilidad Masiva | Seguridad 360 | Fuente Única de Verdad"
- **Core Message:** Transforming data chaos into strategic assets with governance, security, and clean infrastructure
- **Tech Stack:** Snowflake, Databricks, AWS, Azure
- **Key Concepts:** Data lakes, warehouses, petabyte scale, GDPR compliance, single source of truth

### Gemini Prompt
```
1200x630px professional hero image for "Big Data Management" service page.

COMPOSITION:
- Left 60%: Abstract but purposeful visualization of DATA LAYERS/STRATA
  - Horizontal bands representing database layers and data pipelines
  - each layer subtly different shade of teal/blue/gray
  - Conveys "layering", "structure", "governance" without being literal
  - Flowing connections between layers suggest data orchestration
- Right 40%: Minimalist typography area
  - Clean white background
  - Space for headline "Big Data Management" (will be added in code)

STYLE:
- Modern enterprise aesthetic (premium consulting firm, not tech startup)
- Professional photography quality
- Warm lighting, high-key (bright background)
- Accent color: #135c51 deep teal in 15-20% of visual (highlights key layers/nodes)
- Geometric precision: clean lines, deliberate spacing
- NO gradients, NO 3D renders, NO stock imagery

MOOD:
- Confidence: "We handle massive scale professionally"
- Security: Sense of order and control over complexity
- Enterprise: Suitable for CTOs in brewing, logistics, agriculture industries

TECHNICAL NOTES:
- Safe zone: Avoid critical elements in outer 100px margins (for text overlay in design)
- Foreground should be semi-transparent friendly for white text overlay
```

## 📋 Implementation Checklist

After generating each image with Gemini:

- [ ] **Test at 1200×630px** — verify legibility and composition at actual OG image size
- [ ] **Text overlay test** — ensure 40-60% of each image remains clear/bright for text
- [ ] **Brand color validation** — teal (#135c51) should accent 15-20% of image, feel intentional
- [ ] **Consistency pass** — all 5 images should feel like a cohesive set (same lighting, tone, professionalism level)
- [ ] **Mobile preview** — verify images don't look cramped when scaled down on mobile
- [ ] **SEO-ready** — add descriptive alt text matching service titles once deployed
